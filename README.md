# affi-asso

Application de gestion pour association.

## A propos

**affi-asso** est un projet open source destiné à simplifier la gestion quotidienne d'une association : suivi des membres, gestion des activités et communication interne.

## Fonctionnalités prévues

- Gestion des membres (inscription, cotisations, annuaire)
- Gestion des événements et activités
- Communication interne (annonces, notifications)
- Tableau de bord pour les administrateurs

## Démarrage

> La stack technique et les instructions d'installation seront documentées au fur et à mesure du développement.
